Today I started building my personal website as part of the ThinkNeuro Software Engineering Practicum. I picked the Astrofy template because it comes with a homepage, a projects page, and a resume page all in one place.

## What I changed

- Updated the homepage with my name, bio, and current projects
- Added my ASL gesture recognition project and 8-bit computer build to the projects page
- Filled in my resume with my education, experience, and skills
- Updated all the social links to point to my own GitHub and email

## What I learned

Working through this taught me the basics of cloning a repo, running a local dev server, and editing Astro components. It also gave me a reason to organize all my project write-ups in one place instead of leaving them scattered across GitHub and Medium.

More posts coming as I keep building this out.
